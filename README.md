# getnada-random-email

generate random email dari getnada dan sekaligus mengecek isi email yang di dapatkan.

cara pakai :
1. install xampp
2. jalankan command : php email.php
3. enjoy!

semoga berguna dan jangan lupa klik tombol stars nya.
